使用迴圈方式呈現所有 book資料

    {% for book in books %}
  • id: {{ book.id }}
    • title: {{ book.title }}
    • is_published: {{ book.is_published }}
    • {% endfor %}

使用迴圈搭配if判斷，只顯示 is_published = True 的資料

    {% for book in books %} {% if book.is_published %}
  • id: {{ book.id }}
    • title: {{ book.title }}
    • {% endif %} {% endfor %}
